Detailed Description

This class represents a single recorder which can either playback or record to/from a file.

The recorder records transformation, audio and generic data individually and write recorded data to file. For playback the individual files are read and data is returned.

Records generic buffer data for a certain id and a certain point in time.

In this method data is moved to a DTO which is then added to a recording buffer. As soon as the buffer extends a maximum size the buffer data is moved to a writing buffer. Then a thread task for writing the data from the writing buffer to file is created.

Parameters
time_stampThe point in time for which data is recorded
idThe id for which data is recorded (should be unique for different types and instances of data sources)
int_aA buffer of ints that can contain semantic data (e.g. the current color of a object, the state of a process, ...)
float_aA buffer of floats that can contain semantic data (e.g. the current distance between users, the current network latency, ...)
char_aA buffer of characters that can contain semantic data (e.g. the name of an event, ...)
Returns
true if successful, false otherwise

Records audio buffer data for a certain sound source and a certain point in time.

In this method data is moved to a DTO which is then added to a recording buffer. As soon as the buffer extends a maximum size the buffer data is moved to a writing buffer. Then a thread task for writing the data from the writing buffer to file is created.

Parameters
sound_dataThe buffer containing the audio data
sound_data_lengthThe size of the audio buffer (should be smaller than the size of the audio buffer of a SoundDTO
sampling_rateThe sampling rate of the sound source
start_indexPotential offset index from which on new data is present (if 0 all data is new, if n>0 then the first n samples are not recorded)
channel_numThe number of channels of the audio source (Note that unity records audio in an interleaved manner e.g. for 2 channel audio L0R0L1R1...)
time_stampThe timestamp corresponding with the start time for the first sound sample
sound_idThe id referencing the sound source
Returns
true if successful, false otherwise
