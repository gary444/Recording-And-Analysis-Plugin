Returns the version number of the plugin.

This function can be used to retrieve information about the current plugin version. This can be crucial because data formats might be different between plugin versions potentially leading to crashes when recordings created with different version should be played back.

Parameters
textstring into which the version number is written
max_lengthmax length of the string
Returns
length of the string or -1 if the string would exceed capacity

channel_numnumber of channels of the sound contained in the array. Note, that data is assumed to be in interleaved fashion e.g. for 2 channel C1S0,C2S0,C1S1,C2S1, ...
time_stamppoint in time for which data is recorded
sound_idid of the sound source
